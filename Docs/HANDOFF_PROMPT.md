## 🎯 核心特性

### 1. 三维记忆系统
- Factual Memory（事实记忆）
- Experiential Memory（经验记忆）
- Working Memory（工作记忆）

### 2. 智能场景判断
- 自动识别新项目/现有项目
- 智能选择初始化模式
- 开箱即用体验

### 3. 多代理协作
- 17个专业代理
- 按需调度和协调
- 并行执行能力

### 4. Hooks自动化
- pre-tool钩子（安全检查）
- post-tool钩子（自动格式化）
- pre-commit钩子（测试验证）
- session钩子（环境检查）
- task钩子（任务管理）
- notification钩子（桌面通知）
